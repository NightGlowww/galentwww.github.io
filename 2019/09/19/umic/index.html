UmiMiCraft 链接指南

2019年09月19日 1.5k 字 大概 6 分钟 来玩儿啊 Minecraft

更新日志、说明

来一起愉快的玩儿 Minecraft 吧~

（非常重要）本文于 8 月 11 日更新了服务器地址和增量一 Mods 包，请悉知！

（基本没用）本文于 8 月 12 日重写了文章结构，请悉知！

【08.11】更新了数个 mod ，增量更新文件下载地址增量一，请解压并替换到游戏 Mod 文件夹。

基础包在这里：下载地址；下载过基础包后还需要下载【增量一】

你好！

本文将指导你如何快速链接至 UmiMiCaft 服务器！这可能和您之前听到过的不太一样，一起来看看吧~

准备工作

和大部分 Minecraft 客户端一样，在使用之前请您确保已经安装了 64 位的 Java 8 吧，如果还没有安装 Java 8 的话，启动器和游戏是没办法启动的！请悉知！

我们自然是推荐您前往 Oracle 官网下载并安装”Java SE Runtime Enviroment 8” 并附上

下载地址链接

但由于国内网络环境的原因或是现在部分的下载要求注册 Oracle 账户，我们也提供一个国内云盘下载链接

蓝奏云下载地址链接

随后跟随指引安装完成 Java 即可，如无必要，请不要改变 Java 的安装路径。

当你安装完成了 Java 之后，您便可以打开我们提供的 HMCL 启动器了，我们接下来的游戏都是用 HMCL 启动器管理文件，或进行登录

HMCL分流下载地址

HMCL 位于 Github 上的项目页

    账号注册

    为了照顾没有正版 Minecraft 的用户，我们使用了目前使用人数最多的外置验证解决方案——统一通行证，若您不太会使用统一通行证，接下来我们将会简短的提示您如何使用，若您已经知晓了如何使用，请看下一部分。

    注册

    使用注册地址进行注册

    您可以看到：

    • 用户名：是您登陆统一通行证后台的凭据；
    • 密码：是您登录统一通行证后台的密码；
    • 游戏 ID：是您在游戏中别的玩家看到的名字

填写完毕信息后，点击确定并加入，您就拥有了一个统一通行证了，之后您在访问其他的支持统一通行证的服务器，就可以使用这个了。

登录到账号操作页面

您可以在这里完成邮箱绑定，皮肤更换

统一通行证的登录地址如下：统一通行证打开的页面如下

在这里输入您之前注册的用户名和密码，切换到用户后台

若系统提示您先绑定邮箱则绑定邮箱，没有提示的，在页面右上角的个人 ID 处能够找到绑定按钮

这个 QQ 邮箱地址将作为您配对客户端的凭证，请选用一个能接到邮件的常用邮箱。

这样必要工作就准备好了。

更换皮肤

我们完全不建议您使用默认的 Steve 和 Alex 皮肤，这可能导致您无法登录或是其他的问题，请您遵循此步骤进行皮肤更换！谢谢配合。

您可以在 MCBBS 上下载到您喜欢的皮肤；对于文章更新前注册的，且还没有更换皮肤的用户，我们将为您随机置入皮肤，您也可以跟随此步骤进行修改。

在首页点击服务器下方的更换皮肤按钮

随后会打开您在 UmiMiCraft 的皮肤管理器，上传您喜欢的皮肤，选择您喜欢的身形，点击确认即可

UmiMiCraft 支持您使用自定义披风，您也可以在这里一并上传。

这样账号配置的所有工作就做完了

配置启动器

安装游戏

UmiMiCraft 使用 Minecraft 1.12.2 作为主程序版本，对于老用户可以自行下载对应版本。

账户联通

新 HMCL 用户的教程如下：

打开 HMCL 会弹出如下提示

在【登录方式】中选择【外置登录】

并在认证服务器中点击右边的加号填写服务器的验证地址

https://auth2.nide8.com:233/be71bcb4b2b511e991d7525400b59b6a/

然后在【用户名】处填写您绑定的 QQ 邮箱，【密码】填写您在刚刚注册的统一通行证密码即可。

如果均填写正确，会在启动器上显示您的皮肤头像和用户 ID

游戏本体配置

请点击【没有游戏版本】按钮，随后点击右侧上方的添加版本，选择 1.12.2 ，随后安装随附的Forge 和 OptFine，若有版本选择的，均选择最新版本即可。

若提示失败则多试几次，或点击【游戏】—【自动安装】—【在线安装】选择安装 Forge 和 OptFine，直至全部安装成功。

预启动

点击左上角的小房子图标回到主页，点击启动游戏，启动器会自动补齐所需要的文件，请在此过程中保证网络的畅通，当可以进入游戏后，设定语言，然后关闭游戏。

加载 Mods

我们已经将所用到的 Mods 以压缩包的形式上传到云盘，请下载并移动至

游戏路径/.minecraft/mods 中

Mods 下载地址是：下载地址

随后再次启动游戏，本次启动时间将会很长，请耐心等待。

添加服务器

若【多人游戏】选项中没有服务器，请点击添加服务器，服务器名称随意填写，地址请填写

47.92.70.248:25577

随后就可以连接进入服务器了
